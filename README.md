## Description

[Next](https://nextjs.org/) framework TypeScript starter repository.

## Installation

```bash
$ npm install
$ npm install -g json-server
```

## Running the app

```bash
# local json db
$ json-server --watch db.json -p 3333

# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
